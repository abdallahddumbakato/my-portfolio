# Contributing to Ddumba Abdallah Kato's Portfolio

Thank you for your interest in contributing to my portfolio! Here are some guidelines to help you get started.

## How to Contribute

### Reporting Bugs
- Use the GitHub Issues tab to report bugs
- Describe the bug in detail including steps to reproduce
- Include your browser and OS information

### Suggesting Enhancements
- Open an issue describing your enhancement suggestion
- Explain why this enhancement would be useful
- Provide examples if possible

### Code Contributions
1. Fork the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Commit your changes (`git commit -m 'Add some amazing feature'`)
5. Push to the branch (`git push origin feature/amazing-feature`)
6. Open a Pull Request
